# Progress
Прототип блока Progress, реализованный на чистом JavaScript, для использования в мобильных web-приложениях.
Компонент не привязан к верстке страницы, управляется через API, что позволяет легко переиспользовать блок в других проектах.

Демонстрация реализации [*вот тут*](https://viilkaa.github.io/ozon-intership/)

## Что было сделано?
- Векторная реализация прогресса в виде круга
- Управление значением прогресса
- Анимация вращения
- Состояние скрытия элемента со станицы
- Простой API для управления компонентом

## Методы управления
- setValue(75) - установит значение value
- setAnimated(true) - управляет анимацией
- setHidden(true) - скроет элемент со страницы

## Пример переиспользования
Подойдет любой html контейнер

`<div id="progressBar"></div>`

И подключаем компонент в js

`import { progress } from "./progress.js";`
